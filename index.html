Tippster App

Tipster is a React web application that allows users to thank service workers without the hassle of cash or the awkwardness of a physical transaction. The data was organized with postgreSQL and all server-side code executed with Node.js. Financial transactions made possible by the Stripe API, text message recipts via Twilio API, & Google Maps help users to find each other.
